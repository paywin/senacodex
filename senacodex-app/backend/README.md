# SENACODEX Backend

Backend da aplicação SENACODEX - Sistema de Gerenciamento de Projetos Integradores

## Setup rápido

```bash
npm install
npm run dev
```

### Requisitos
- Node.js 18+
- PostgreSQL 12+

### Variáveis de Ambiente

Crie um arquivo `.env` com as variáveis em `.env.example`

### Scripts disponíveis
- `npm run dev` - Iniciar servidor de desenvolvimento
- `npm run build` - Compilar TypeScript
- `npm start` - Iniciar servidor de produção
- `npm run lint` - Executar linter
